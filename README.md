# TEST

A basic test project demonstrating testing infrastructure with Jest.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the main application:
   ```bash
   node index.js
   ```

3. Run tests:
   ```bash
   npm test
   ```

4. Run tests in watch mode:
   ```bash
   npm run test:watch
   ```

## Project Structure

- `index.js` - Main application entry point
- `utils.js` - Utility functions module
- `utils.test.js` - Test suite for utility functions
- `package.json` - Project configuration and dependencies

## Features

This project includes:
- Basic utility functions (add, multiply, isEmpty)
- Comprehensive test coverage with Jest
- Example of testing different scenarios (positive, negative, edge cases)
- Documentation and usage examples
